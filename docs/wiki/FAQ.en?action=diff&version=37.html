    171171 * With boxes with an USB-Host (e.g. 7170,7270) you can externalize some packages on an USB device (e.g. USB Stick, USB hard disk).  The externalize-process is done at the end of the build process via the [wiki:help/howtos/common/external external]-script. At menuconfig, there is an option to do the externalizing. Only predefined parts of packages are being externalized to a USB-Device, as opposed to using USB-Root which will move the entire file system onto the USB drive.
