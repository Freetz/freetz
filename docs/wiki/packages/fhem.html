FHEM

​FHEM ist ein Open Source Programm zur Verwaltung von Geräten zur Home Automation. Der Focus liegt dabei auf Geräten der FS20 Spezifikation von ​Contronics - es werden aber auch weitere Systeme unterstützt. FHEM ist überwiegend in Perl geschrieben, und nutzt den Apache Web Server zur Bereitstellung eines Web-Frontends, mit dem sich die verfügbaren Geräte verwalten und Steuern lassen.

Installation

Eine aktuelle Installationsanleitung findet sich unter ​http://www.koeniglich.de/fhem/fritzbox.html, und setzt auf Freetz auf. Da die dortige Seite vom Entwickler selbst gepflegt (und somit aktuell gehalten) wird, möchte ich diese Informationen hier nicht duplizieren.
